Projects

File Box

File box is an anonymous file storage and retrieval service. When a user submits file, it is compressed on the frontend, passed to the backend, and in return the user receives a 64 character retrieval key that they can enter to get their file back at anytime. This key can be shared with others or kept private. File box allows for efficient and secure sharing and storage of files.

Newc

Newc is a simple language inspired by C, Rust, and TypeScript. For this project I used Rust to write a compiler for the language. Newc is a systems level programming language that compiles to machine code with the help of LLVM. This was a great project for learning how the internals of a compiler work. While I was working through it I frequently referenced the TypeScript compiler implementation.
